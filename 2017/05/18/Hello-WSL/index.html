Hello WSL

Posted on 2017-05-18

最近在看Operating Systems: Three Easy Pieces.
因为source code需要在UNIX系统下编译, 又不想去装VM, 就尝试了一下WSL, 感觉还是挺好用的.

Installation Guide

期间也是踩了几个坑:

  1. 选中Develop Mode之后, 会有一个spinner “Searching for Develop Mode packages”, 一直没有结果. 其实不用管它
  2. Turn on windows feature巨慢无比. 在没完成的情况下我重启了一下机器,它竟然就好了.
  3. 要用管理员权限运行VSCode, 才能在VSCode的terminal里面获得su权限. (据说WSL-terminal比较好用)
